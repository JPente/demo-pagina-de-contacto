### Características Principales
* Diseño Responsivo: Adaptación completa a resoluciones de escritorio, tabletas y móviles mediante Media Queries.

* Validación de Formularios: Implementación de lógica en JavaScript para validar la entrada de datos antes del envío.

* Estética Minimalista: Uso de fondos optimizados y una paleta de colores coherente para mantener la claridad visual.

* Interactividad: Efectos visuales en elementos de entrada y botones para mejorar el feedback táctil/visual.

### Tecnologías Utilizadas
HTML5: Estructuración semántica del contenido.

CSS3: Estilizado avanzado, incluyendo Flexbox y Grid para el posicionamiento.

JavaScript (ES6+): Manipulación del DOM y manejo de eventos del formulario.

### Estructura del proyecto:
│  
├── img/          # Logotipos y fondos de la landing page
├── css/          # Archivos de estilos
├── js/           # Lógica de validación y comportamiento
└── index.html    # Estructura principal
